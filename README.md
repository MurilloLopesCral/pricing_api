# Política de Privacidade – Pricing Analytics GPT

Este assistente (Pricing Analytics GPT) é uma ferramenta interna desenvolvida para apoiar análises de pricing, margem de contribuição, faturamento e indicadores comerciais da empresa.

## Dados processados
O assistente pode processar, sob demanda do usuário:
- Dados agregados de vendas
- Informações de produtos
- Indicadores financeiros (ex.: faturamento, margem, custo)
- Filtros como região, cliente ou período

Os dados são utilizados **exclusivamente para fins analíticos e de apoio à tomada de decisão**.

## Armazenamento e compartilhamento
- O assistente **não armazena dados pessoais permanentemente**.
- Os dados consultados são processados em tempo real via API interna.
- Nenhuma informação é compartilhada com terceiros fora da infraestrutura da empresa.

## Segurança
O acesso à API é protegido por autenticação.
As consultas são limitadas a dados previamente autorizados.

## Uso interno
Este assistente é destinado **exclusivamente para uso interno** por colaboradores autorizados.
